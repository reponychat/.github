# Reponychat


## Описание

Проект, который признан возродить старый и добрый [ponych.at](https://www.youtube.com/watch?v=dQw4w9WgXcQ)

UPD: в связи большого кол-во в опросе не знания проекта *ponych.at*

Поничат - это чат без регистрации, что-то вроде дискорда, тг (последний больше всего описывает его), куда вход в канал происходит под рандомной пони

Фишки, которые были в прежнем чате:

- Чатиться (это понятно)
- Делать действие от персонажа (в основном для ролплея там использовался)
- Кидать кубик (кубики любых граней по формуле `x` раз `y` граней +|- `mod` добавляется или убавляется число)
- ставить видео в очередь из ютуба (чтобы все могли посмотреть)
- наличие других каналов и одного скрытого

## Особенности

- Разработка будет вестись с нуля, потому, технологически данный проект должен оказаться лучше прежнего
- Также по желанию комьюнити может быть добавлена какая-то фича
- Планируется расширить функционал старого чата, но также с учетом мнения комьюнити для сохранения вайба
